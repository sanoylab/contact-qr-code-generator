
# Contact QR Code Generator

This is a simple contact QR code generator. You can provide:
- Full Name
- Email
- Phone number
and generates a contact QR code image.




## Tech Stack

**Client:** HTML, CSS, JavaScript, qrcode.min.js, Bootstrap
